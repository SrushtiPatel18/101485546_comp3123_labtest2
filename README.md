# 123456789_comp3123_labtest2

## Description

This is a simple and user-friendly React Weather Application that fetches real-time weather data based on a city's name.  
The app displays temperature, humidity, weather condition, icon, and location information.  
Users can search any city using the search bar.

## Demo
Live: https://101485546-comp3123-labtest2-git-main-srushtis-projects-70605c48.vercel.app/

## Setup
1. Clone repo
2. `npm install`
3. `npm start`

## API
Using OpenWeatherMap Current Weather API:
http://api.openweathermap.org/data/2.5/weather?q={city}&appid=e06d08657a463d4ca6ac402e40768498

## Notes
- Search by city name
- Displays temperature, feels_like, humidity, wind speed, sunrise/sunset, weather icon.
